CSV Requirements:

The CSV file must contain the following headers (case-sensitive):

  • ID: Unique student ID (e.g., 00001) or leave blank to auto-generate.
  • Name: Student's full name (required).
  • Class: Class/grade (e.g., 10A, required).
  • Roll: Roll number (optional).
  • Mobile: Mobile number (optional).
  • Year: Academic year (e.g., 2025, optional).

Example CSV format:

ID,Name,Class,Roll,Mobile,Year
00001,Test User 1,10A,101,1234567890,2025
,Test User 2,10B,102,9876543210,2025
